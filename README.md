# SpaceX Company Profile Demo

Static company profile prototype for GitHub Pages.

## Local preview

```powershell
python -m http.server 8000
```

Open:

```text
http://127.0.0.1:8000/
```

## GitHub Pages

Publish from the `main` branch root. The default entry is `index.html`.

For another coding agent, read AGENT_GUIDE.md first. It explains the project structure, data contract, rendering notes, and GitHub Pages workflow.
